About MeTL

MeTL is openSource software, protected by the Apache 2 Open Source License. The source code is available at: https://github.com/StackableRegiments/analyticalmetlx

MeTL uses a number of openSource libraries and online services.

CloudConvert

https://cloudconvert.com

MeTL uses CloudConvert to parse a variety of document types, to import into the conversations. MeTL complies with the CloudConvert terms of usage

TokBox

https://tokbox.com

MeTL uses TokBox to provide videoconferencing within a conversation. MeTL complies with the TokBox terms of usage. Videostreams generated by MeTL's videoconferencing are covered by TokBox's privacy policy.

Liftweb

https://github.com/lift/framework

MeTL's backend server is built on Lift.

MeTL uses Lift, under the terms of the Apache 2.0 license.

IMSGlobal basiclti-util-java

https://github.com/IMSGlobal/basiclti-util-java

MeTL uses basiclti-util-java from IMSGlobal library to comprehend LTI launch requests from LMS tools.

MeTL uses basiclti-util-java, under the terms of the Apache 2.0 license.

Vysper

https://mina.apache.org/vysper-project/

MeTL uses Vysper to provide XMPP protocol support.

MeTL uses Vysper, under the terms of the Apache 2.0 license.

PAC4J

https://github.com/pac4j/pac4j

MeTL uses PAC4J to provide SAML authentication and authorization support.

MeTL uses PAC4J, under the terms of the Apache 2.0 license.

Google API client library

https://developers.google.com/api-client-library/java/

MeTL uses Google API client library for java to provide OpenID connect authentication support.

MeTL uses the Google API client libraries for Java, under the terms of the Apache 2.0 license.

POI

https://poi.apache.org/

MeTL uses Apache POI to comprehend Microsoft Office document formats.

MeTL uses Apache POI, under the terms of the Apache 2.0 license.

EHCache

http://www.ehcache.org/

MeTL uses EHCache for performance optimization.

MeTL uses EHCache, under the terms of the Apache 2.0 license.

H2 Database

http://www.h2database.com/

MeTL uses H2 Database, under the terms of the dual MPL 2.0 and EPL 1.0 license.

http://shiro.apache.org/

MeTL uses Apache Shiro, under the terms of the Apache 2.0 license.
